# 用户画像调整功能说明

## 功能概述

在全域洞察（VOC洞察）页面中，新增了用户画像调整功能。用户可以点击每个画像，通过CDP标签库进行精准筛选和调整。

## 功能特性

### 1. 用户画像列表
- 位置：VOC数据面板中部
- 显示内容：
  - 画像名称（如：科技精英、家庭主力等）
  - 匹配用户数（users）
  - VOC文本数量（VOC）
- 交互：点击任意画像卡片，弹出CDP标签库筛选框

### 2. CDP标签库筛选框
- 弹窗尺寸：900px宽，最大高度80vh
- 标签分类（6大类）：
  1. **人口统计**：18-25岁、26-35岁、36-45岁、46-55岁、56岁以上
  2. **收入水平**：10万以下、10-30万、30-50万、50万以上
  3. **职业类型**：科技行业、金融行业、教育行业、医疗行业、政府机关、企业主
  4. **生活方式**：家庭导向、事业导向、户外运动、科技爱好者、环保主义者
  5. **购车行为**：首次购车、增购换购、豪华品牌偏好、纯电优先、安全优先
  6. **用车场景**：日常通勤、家庭出行、商务接待、长途旅行、城市代步

### 3. 实时数据更新
- 选择标签时，底部实时显示：
  - 已选标签数量
  - 匹配用户数（动态计算）
  - VOC文本数量（动态计算）
- 确认调整后，画像卡片上的数据同步更新

### 4. 预设画像数据
系统预设了5个典型用户画像：
1. **科技精英**：342 users, 856 VOC - 26-35岁、30-50万、科技行业、科技爱好者
2. **家庭主力**：289 users, 723 VOC - 36-45岁、10-30万、家庭导向、家庭出行
3. **环保先锋**：156 users, 412 VOC - 26-35岁、30-50万、环保主义者、纯电优先
4. **商务精英**：198 users, 534 VOC - 36-45岁、50万以上、企业主、商务接待
5. **安全至上**：262 users, 678 VOC - 46-55岁、30-50万、安全优先、家庭出行

## 使用流程

1. 进入"全域洞察"标签页
2. 在左侧VOC数据面板中找到"用户画像"区域
3. 点击任意画像卡片（如"科技精英"）
4. 在弹出的CDP标签库中勾选/取消标签
5. 底部实时查看匹配的用户数和VOC文本数
6. 点击"确认调整"保存更改，或点击"取消"放弃更改
7. 画像卡片上的数据自动更新

## 技术实现

- **状态管理**：使用React useState管理画像数据和选中标签
- **弹窗组件**：固定定位的模态框，带遮罩层
- **标签选择**：多选checkbox，支持跨分类选择
- **数据计算**：基于选中标签数量动态计算匹配用户数（模拟算法）
- **响应式布局**：标签库采用2列网格布局，适配不同屏幕尺寸
